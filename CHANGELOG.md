# Changelog

All notable changes to Matrix Screensaver Pro will be documented in this file.

## [1.12.1] - 2025-12-14

### Added
- Initial public release
- DirectX GPU-accelerated rendering at 60 FPS
- Multiple theme presets (Classic Green, Blue Matrix, Red Alert, Purple Haze, Rainbow)
- Random colors mode with vibrant per-column colors
- Optional clock display with date and time
- Mirror mode with occasional upward-flowing columns
- Easter eggs with Matrix quotes
- Live settings with real-time preview
- Multi-monitor support
- Built-in log viewer
- Custom color selection
- Adjustable font size (10-30px)
- Variable scroll speed and density controls
- Katakana and Extended ASCII character sets
- Bold characters and glow effects options

### Features
- Self-contained installer (includes .NET runtime)
- Start Menu shortcuts for easy settings access
- Settings persistence across sessions
- Smooth animations and transitions

### System Requirements
- Windows 10/11 (64-bit)
- DirectX 11 compatible GPU
- 100 MB RAM minimum

## Release Notes Format

Versions follow [Semantic Versioning](https://semver.org/):
- **MAJOR** version for incompatible changes
- **MINOR** version for new features (backwards-compatible)
- **PATCH** version for bug fixes

### Change Categories
- **Added** - New features
- **Changed** - Changes to existing functionality
- **Deprecated** - Soon-to-be removed features
- **Removed** - Removed features
- **Fixed** - Bug fixes
- **Security** - Security updates
